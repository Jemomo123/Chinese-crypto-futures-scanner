# Crypto Market Scanner

## Overview
A mobile-friendly crypto market scanner that detects expansion and TC20 patterns across multiple timeframes.

## Features
- Multi-timeframe analysis (15m, 1h, 4h)
- Expansion pattern detection
- TC20 pattern detection
- Mobile-friendly HTML reports
- State management per symbol
